Le Sumobot

  1. 0 - Comment se passe le cours

    Ce chapitre n'est pas obligatoire mais permet de savoir comment mieux utiliser ce cours.

  2. 1 - Le Cahier des charges

    Ce chapitre est nécessaire pour savoir ce que nous allons faire et contient tout les prérequis.

  3. 2 - L'Atmega 328p ou mini Arduino uno

    Ce chapitre permet de vous enseigner comment construire votre propre carte Arduino minuaturiser.

  4. 3 - Connexion Wire

    Grâce à ce chapitre, vous réussirez à connecter les cartes que vous venez de fabriquer dans le chapitre précédent.

  5. 4 - Le bouton

    Ce chapitre permet d'apprendre à utiliser un bouton, votre premier capteur de ce projet.

  6. 5 - L'acceleromètre

    Ce chapitre vous permet d'utiliser un accelerometre, indispensable pour savoir si le robot a été pousser.

  7. 6 - Le capteur infrarouge, pseudo : TCRT5000L

    Ce chapitre va vous permettre grace à une led infrarouge, des résistances et des capteurs infrarouges de détecter les lignes blanches et noires.

  8. 7 - Les capteurs à ultrasons

    Grâce à ce chapitre, vous mesurerez les distances tel une chauve-souris.

  9. 8 - Les moteurs

    Vous attendiez ce chapitre, comment imager une voiture sans moteurs mais celle-ci aura des roues-omnidirectionnelles.

  10. 9 - Le servomoteur

    Le sermoteur va vous permettre de relever l'avant du robot mais aussi vous apprendre à utiliser une librairie.

  11. 10 - L'alimentation

    Dans ce chapitre, Arduino ne sera pas utiliser mais l'alimentation grâce à deux piles 9V est nécessaire.

  12. 11- La coque en 3D

    Ce chapitre va vous expliquer comment modéliser la coque tout en restant dans les limites de tailles : 10cm x 10cm.

  13. 12 - Le code

    Le chapitre le plus long et qui va surement vous apprendre le plus de choses : coder ce sumobot de manière concise et ordonner.
