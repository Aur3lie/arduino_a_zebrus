1. Utilisation

_______ Comme vous avez pu le comprendre, la place nous manque dans ce projet. Au lieu de nous encombrer avec une carte Arduino déjà toute faite, nous allons créer notre propre carte Arduino. Ne vous inquiétez pas, c'est plus simple que ce que vous croyez.

2. Composants

_______ Nous allons avoir besoin de divers composants que je vais ici vous énumérer et en expliquer le fonctionnement. Le premier est l'atmega 328p qui est le microcontroleur. Cet atmega a le même nombre d'entrées/sorties que la carte Arduino. C'est autour de ce composant principal et essentiel que vont se brancher tous les autres composants. Il vous faudra aussi un quartz de 16MHz qui comme son nom l'indique oscille à 16MHz et qui est l'horloge de la carte. Il vous faudra aussi une resistance et autres condensateurs mais aussi une carte Arduino Uno et un FTDI pour le téléversement.

3. Flasher le bootloader

_______Tout d'abord avant de flasher le bootloader, essayons de savoir ce que cela signifie. Pour schématiser, un bootloader est une sorte de système d'exploitation et flasher est en quelque sorte l'installation de ce bootloader. Donc notre atmega arrive sans bootloader, il faut donc réaliser le schéma ci-dessous puis connecter votre carte Arduino à votre ordinateur. Ensuite, vous pourrez ouvrir le logiciel Arduino et aller dans fichiers --> exemples --> ArduinoISP --> ArduinoISP, puis dans outils --> Programmateur --> Arduino as ISP et enfin cliquez sur outils --> graver la séquence d'initiatialisation. Avant de faire tout cela, soyez certain d'avoir choisi le bon port COM et le bon type de carte (Arduino Uno).

4. Réalisation de la carte finale

Nous avons pu voir ci-dessus que le principe de fonctionnement était compliqué, y a-t-il une bibliothèque pour nous aider ?

_______Maintenant que vous avez fini de flasher le bootloader, vous pouvez réaliser le schéma suivant.

_______Ensuite, vous pouvez utiliser votre carte comme une carte Arduino Uno et utiliser le FTDI pour téléverser le code. N'oubliez pas de changer le programmateur de la carte de "Arduino as ISP" en " AVRISP mkll". Et voilà vous avez réussi ce qui prouve que ce n'était pas si dur que cela.

            #include <Servo.h>
            
            Servo monServo
            
            void setup() {
              monServo.attach(11);
            }
            
            void loop() {
              // put your main code here, to run repeatedly:
              monServo.write(0);
              delay(500);
              monServo.write(180);
              delay(100);
            }
